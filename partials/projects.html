Romantic-Era Lyrics

September 2013 - December 2015

A website and database of English Romantic-Era sheet music and poetry.

The development version is stored on my server, and the source code will be available by the end of this year. I've been responsible for most of the planning and design involved in this project, and I was responsible for all front-end and back-end development.

I also managed and trained a team of student interns from the USC Honors College tasked with data-entry, data-collection and reformatting existing data to make it more compatible with the database schema.

  • Backend implemented with PHP and MySQL
  • Frontend built using AngularJS

Critical Interactives: Ward One

January 2015 - May 2015

An iPhone application and accompanying dynamic website that will present the story of the predominantly African American community in Columbia, South Carolina known as Ward One.

I worked on this project through a class at USC. This is an application that will allow users to walk through the area of Columbia formerly known as Ward One and see what was once there.

I was involved in planning the app and designing the backend framework and organization using a Django project. I determined that it would make the most sense to store our data of people and places on a server because the application will eventually have an accompanying website.

I also made the project website.

Muntaser Arabic Dictoinary

October 2014

A simple Arabic/English and English/Arabic dictionary.

I built this before I began the Romantic-Era Lyrics project to give myself some familiarity with designing a database with MySQL, populating it with a PHP script, building a PHP API, and using that API with AngularJS.

  • Searchable in both English and Arabic
  • Also searchable with or without vowels in Arabic
  • Built using PHP, MySQL, and AngularJS
